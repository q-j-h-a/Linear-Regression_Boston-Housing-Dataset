1. 数学原理

导数与偏导数

导数可以理解为某一点的“斜率”，它反映了函数值变化的快慢。

偏导数可以这样理解：如果你站在一个不平的平面上， 只沿着某一个方向（例如 X 或 Y）移动，那么对应的变化率就是偏导数。

在模型训练中，我们利用导数来判断：参数应该往哪个方向调整，误差会下降得更快。

均值与标准差

  • 均值（Mean）：数据的平均水平
  • 标准差（Standard Deviation）：数据的波动大小

由于不同特征的数值范围差别较大（例如面积和房间数）， 如果不处理，会影响模型训练效果。

通常需要对数据进行标准化处理，使不同特征处于相近的尺度。

2. 机器学习基础

监督学习

监督学习可以理解为：给模型提供“输入”和“正确答案”，让它学习两者之间的关系。

  • 输入：房屋的各种信息
  • 输出：对应的房价

模型学习完成后，就可以对新的数据进行预测。

特征与标签

  • 特征（X）：影响结果的因素，例如面积、房间数等
  • 标签（y）：需要预测的目标，例如房价

回归与分类

  • 分类：结果是离散的类别（例如猫或狗）
  • 回归：结果是连续数值（例如房价）

本实验属于回归问题，使用的是线性回归方法。

3. Python 编程基础

NumPy

在处理大量数据时，使用 NumPy 的向量化运算可以提高计算效率， 避免使用低效的 for 循环。

Matplotlib

散点图

用于展示数据分布，例如用横轴表示房间数，纵轴表示房价。

直线图

用于绘制模型拟合的结果，即一条尽可能贴近所有数据点的直线。

线性回归的目标，就是找到这样一条“最合适的直线”。
