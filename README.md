# Linear-Regression_Boston-Housing-Dataset

波士顿房价预测教学实验项目，当前应用目录是 `simple_linear_regression`。

## 运行方式

推荐使用 Python 3.10 或更高版本。

从仓库根目录启动：

```bash
cd simple_linear_regression
python3 -m pip install -r requirements.txt
python3 app.py
```

也可以在仓库根目录直接运行脚本，数据文件会按 `app.py` 所在目录读取：

```bash
python3 -m pip install -r simple_linear_regression/requirements.txt
python3 simple_linear_regression/app.py
```

如果当前已经在项目目录中：

```bash
python3 -m pip install -r requirements.txt
python3 app.py
```

启动后访问：

```text
http://127.0.0.1:5000/
```

更多功能说明见 `simple_linear_regression/README.md`。
